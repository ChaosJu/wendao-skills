## 1. 2018年"录音门"事件（最核心争议）

### 1.1 事件经过【二手-权威】

- 2018年7月3日深夜至7月4日凌晨，一段时长约**50分钟**的李笑来私下谈话录音在微信、媒体疯传，引爆币圈。
  - 来源：新京报《李笑来变脸：从人生导师到韭菜之王》 http://www.bjnews.com.cn/invest/2018/07/05/493926.html
  - 来源：21世纪经济报道 https://m.21jingji.com/article/20180704/herald/107797937590e2d22d9c59f55935a806_zaker.html
- 录音录制于 **2018年春节前后（2月4日）**，背景是几位币圈从业者（厦门区块谷总经理吴子龙夫妇等）向李笑来"请教经验"，被对方偷偷录制，约5个月后流出。
  - 来源：新浪科技 https://tech.sina.cn/it/2018-07-13/detail-ihfhfwmu8561905.d.html

### 1.2 录音里到底说了什么【一手-录音转写 / 二手-权威整理】

录音言辞极其粗俗。新京报统计：50分钟内 **24处"傻X"、24处"妈的"**。核心内容：

**(a) 对币圈大佬/项目的点评（指名道姓）：**
- **Ripple（瑞波）/孙宇晨**：称 Ripple "从一开始就是傻X项目……核心团队都走光了"，孙宇晨是"忽悠/骗子"。
- **量子链（Qtum）/帅初**："我是第一个给他卖空气币的人，我们给他卖了六个月的空气币，不然他哪有今天……他真是一个牛逼的忽悠者，长得也帅，四处走场。"
- **以太坊**：称其暴涨"是央行干的"——央行要求交易所比特币不能提现，流动性跑到以太坊，"这些都不是什么社区、领导力，不是的"。
- **币安/赵长鹏、何一**：称币安是"骗子交易所"，崛起靠"蹭政策便利"。
- **徐明星（OKEx）**："有黑历史"；**老猫**"曾经是 nobody"；**赵长鹏**"人品不好"。
  - 来源：21世纪经济报道、新京报、经济观察网 https://www.eeo.com.cn/2018/0705/331710.shtml、36氪 https://36kr.com/p/1722639892481

**(b) 自曝"网红/流量收割"方法论（争议最大的部分）：**
- "我实际上是一个网红。我有很大的读者群……谁像我一样有100万读者？所以一瞬间就有爆发。"
- "熊市你又赚不了钱，你就攒人气……次年罗振宇帮我卖书，马上我又跟他合作开专栏。等大牛市来了的时候，我影响力变得比原来还大。这里的核心竞争力是人流。接下来是什么？你怎么去忽悠这件事，你不要把忽悠理解为贬义。"
- "得屌丝者得天下，散户最牛逼，别看你现在天天骂散户这帮傻逼，就是他们供养你。"
- "项目不值钱，但是信的人那么多，到最后就值钱了。"
- "傻X的共识也是共识"；"盲目相信价值投资的都是傻X……赚到钱的才是成功，核心目标只有一个：赚钱。"
  - 来源：经济观察网、新京报、英文全文翻译 Medium《Li Xiao Lai Recording Scandal [Full English Translation]》 https://medium.com/the-birb-nest/li-xiao-lai-recording-scandal-full-english-translation-96dc0e5cd2dd

**(c) 涉及人物名单**（录音点到）：老猫、帅初(NEO/量子链)、以太坊、Ripple、易理华、罗振宇、孙宇晨、赵长鹏、徐明星等。

### 1.3 外界解读【二手-权威】

- 主流定性：录音"揭开了币圈炒作和'割韭菜'内幕的典型样本"。新京报直接给他贴上"韭菜之王"标签，称其完成了从"人生导师"到"韭菜之王"的变脸。
- 36氪《李笑来错了吗？》：相对中立，指出录音揭示的是整个行业真相——"价值来自韭菜的共识，而韭菜的共识来自认为某币会涨的信念。稍不留意，币圈就会滑向传销或者诈骗。"——这条把李笑来个人争议上升为行业结构问题。
- 被点名者的反应很值得玩味：
  - 孙宇晨微博："听说我又躺着中枪了"，热评回复"没事，大家都知道你是骗子"。
  - 何一（币安）："骗子看谁都是骗子，傻X看谁都是傻X……我深刻认识了一下自己傻X的地方，并决定坚持傻X。"
  - 来源：21世纪经济报道

### 1.4 李笑来本人的回应【一手】

李笑来于7月4日先在微博暗示（"我私下里说话脏话很多"），7月6日在个人微信公众号发长文正式回应，核心辩解：

1. **录音是被偷录、恶意传播**：指认厦门区块谷吴子龙夫妇2月偷录、5个月后交给"别有用心的人"，并称对方雇水军、花钱找微博大号转发、组织"维权群"、唆使举报。【一手，但属单方指控】
2. **"我从没说过'韭菜'二字"**：他强调录音里说"韭菜"的是对话另一方，不是他；并悬赏"第一个在录音里找到我说'韭菜'的人，送两万块"。
3. **"没有骂散户傻逼"**：原话是"**不要骂散户傻逼！散户最牛逼**"，被断章取义。
4. **"空气币"是带引号的**：指量子链当时主网未上线的代币状态（他说话时还比了双引号手势），不是说它是诈骗。
5. **"价值投资"原话是"不要盲目相信价值投资"**："盲目"二字被忽略，句意天壤之别。
6. **营销"三步走"是吴子龙提的，不是我说的**。
   - 来源：香港商报 https://hkcd.com/content/2018-07/06/content_1095167.html、网易转载 https://m.163.com/money/article/DM25BINP00258105.html、《韭菜的自我修养》原文 https://0xo.net/745

### 1.5 《韭菜的自我修养》——书面回应【一手】

- 这本小书是李笑来对录音门的延伸回应，开篇即重申"录音里说'韭菜'的是对方，我没说过"。
- 全书把"韭菜"重新定义为"在交易市场不懂还乱来、踏错节奏的散户"，并提供"自我修养"建议（定投、不被情绪左右、认识周期等）。
- **批评者视角的矛盾点**【推断+二手】：书名和"73韭菜节"（他后来把7月3日录音门纪念日命名为"韭菜节"并在 Mixin 上开付费讲座）被批评者视为"一边说自己从不用'韭菜'一词，一边靠'韭菜'这个IP继续变现"，自相矛盾。
  - 来源：非小号百科 https://zh.feixiaohao.wiki/李笑来（记载"2019年7月3日李笑来纪念录音门创'73韭菜节'，在Mixin上开办付费讲座"）

> **矛盾保留**：李笑来坚称"韭菜"不在自己语言体系里；但他随后用"韭菜"出书、办"韭菜节"、做付费社群。这一矛盾是理解他公关策略的关键——把负面词汇收编为自己的品牌资产。

## 2. ICO / 发币争议（EOS、PRESS.one、云币网、INBlockchain/硬币资本）

### 2.1 EOS【二手-权威】

- 2017年6月，李笑来站台/参投的 EOS 发布白皮书，**5天内募资约1.85亿美元**；7月2日二级市场市值冲到近50亿美元，被讥为"价值50亿美元的空气"。
- 质疑点：
  - EOS 众筹**屏蔽美国用户**（美国 ICO 执法严，中国当时监管薄弱）——被指"联合外国人圈中国人的钱"。
  - 募资无上限、用途不明，被指可"零成本自买自卖"操纵价格。
  - 7月3日最高36.58元后一路下跌，到8月29日跌至8.74元，**跌幅约76%**。
  - 来源：凤凰财经 https://finance.ifeng.com/a/20170828/15615891_0.shtml、中国网财经 http://finance.china.com.cn/industry/renwu/20170906/4380052.shtml、中国电子银行网 https://www.cebnet.com.cn/20170821/102417948.html、AiCoin《暴跌76%的EOS是资金盘吗？》 https://www.aicoin.com/zh-Hans/article/44456
- **EOS项目方（Block.one）切割**：2017年9月4日中国"94监管"当晚，EOS团队紧急声明，称李笑来"与EOS项目无关，并非项目联合创始人、董事或高级人员"，并停止接受中国公民购买。【二手-权威，界面/中国网】
  - 矛盾点：业内人士却指出"EOS是Block.one旗下产品，而李笑来等是Block.one团队成员"——切割声明与外界认知相互打架。

### 2.2 PRESS.one（PressOne / PRS）—— 争议升级【二手-权威】

- 2017年7月10日发起，**连白皮书都没有**，仅官网几百字介绍就募资。项目方理由："不提供那个，即使提供了也没多少人看得懂。"
- 拟发售220亿代币，其中100亿枚 PRS 众筹，目标价值2亿美元。实际据ICOINFO数据募得约5848.9个BTC、106270个ETH、3020个EOS，按当时行情约**1.25亿美元**（未达2亿目标）。
- 被广泛斥为"骗局""收割韭菜"的标志性案例——"在'项目书'都没有的情况下，就敢空口筹集2亿美元"。
  - 来源：凤凰财经、界面新闻 https://www.jiemian.com/article/1605261.html、中国电子银行网
- 后续：PRS 长期被批"圈钱后无产品落地"。【推断/二手-一般】

### 2.3 云币网（交易所利益冲突）【二手-权威】

- 投资者质疑：李笑来是**云币网股东**，"利用云币网平台为EOS众筹，相当于把云币网客户的钱变成自己口袋里的钱"——典型的自融、利益冲突指控。
  - 来源：中国网财经、界面新闻
- 概括性指控（来自界面引述业内）：**"李笑来一个人拿到了ICO币圈1/3以上的收入，在他的强力抽血之下，ICO在中国演变成大规模集资诈骗。"**【二手转述业内观点，需谨慎】
- 2017年8月30日，李笑来投资的 ICO 平台 ICOINFO 暂停 ICO 业务；94监管后云币网、ICOinfo率先关闭退出。

### 2.4 "一鸡三吃"指控【二手-一般】

- 中国电子银行网引述网民说法：李笑来通过资本市场"一鸡三吃"（同时做发币方/交易所/站台者三重角色），"成功洗劫了所有人"。
- 李笑来对"骗局"的反驳【一手】："有人说EOS是'五十亿空气'，（但）空气是没有代码的……"

### 2.5 雄岸基金（杭州，政府站台争议）【二手-权威】

- 李笑来任管理合伙人的"雄岸基金"是有杭州市政府背景的百亿基金。被陈伟星指控"利用政府站台概念忽悠粉丝、伪造'政府基金'公众印象（实际政府一分钱没出，只募了几千万）"。
- 李笑来于2018年7月9日宣布辞去雄岸基金管理合伙人职务，并称将起诉陈伟星诽谤。
  - 来源：澎湃新闻 https://www.thepaper.cn/newsDetail_forward_2252976、钛媒体 https://www.tmtpost.com/3337654.html

## 3. 传销 / 资金盘争议

### 3.1 "粉丝金融"模式被质疑【二手-权威】

- 新浪财经《"粉丝金融"的圈钱套路》：李笑来"酷爱集资"，通过基金、股权投资、付费会员"连续开了三个盘子"。
- 其公众号文章《粉丝经济过时了，我们的终点是粉丝金融》：会员前100人年费1000元，第101人起涨到3000元，逐步上调至2万元——递增定价被批评为典型的"早进早赚"盘面话术。
  - 来源：新浪财经 http://finance.sina.com.cn/roll/2018-07-27/doc-ihfvkitx7515599.shtml
- "首富还是首负"之问：新浪指其资产成谜，到底是"首富"还是"首负"外界不得而知。

### 3.2 定投课堂 / BOX / b.watch【一手 + 二手】

- 2019年起，李笑来推"笑来定投课堂"→"定投人生课堂"→"富足人生社群"，并推出 **BOX（b.watch，自称"地球上第一个链上区块链ETF交易所"）**。
- 【一手】李笑来自述 BOX "不收管理费、持仓透明、自己也定投，以身作则"，定位为带普通人"场外赚钱通道"，**只接受比特币支付**，学员可通过销售课程赚比特币。（见 GitHub regular-investing-in-box/Z-Appendix.04.md https://github.com/xiaolai/regular-investing-in-box）
- **批评者/推断视角**：
  - "只接受比特币支付 + 学员卖课赚币"的结构，被部分人认为带有 MLM（多层分销）色彩。【推断，需谨慎，无定罪证据】
  - 在《定投》文本里他自己写道"要么主流资产、要么传销资产"，承认传销资产"短期内可能很赚钱"，只是因不能长期才不碰——这种表述被批"对传销盘的危害轻描淡写"。【一手原文 + 推断】

### 3.3 "BOX"被冒名诈骗【二手-权威，重要澄清】

- **注意区分**：澎湃新闻报道过一个跨国虚拟币诈骗团伙用名为"BOX"的假交易平台行骗（涉案800余万元）。
  - 来源：澎湃 https://www.thepaper.cn/newsDetail_forward_11140309
- 这是**骗子冒名/同名平台**，与李笑来 b.watch 的 BOX 是否同一事物，现有材料无法证实关联，**不应混为一谈**。此处记录是为防止误判。【推断/澄清】

> **矛盾保留**：李笑来一边猛烈批评币圈传销盘、把"传销资产"列为不碰，一边自己的"粉丝金融/递增年费/卖课赚币"模式又被批评者指为类传销结构。两种叙事并存，无司法定性。

## 4. "骗子/大忽悠"公众形象 + 知识付费批评

### 4.1 双重身份的撕裂【二手-权威】

- 李笑来的公众形象长期两极：粉丝奉为"人生导师/财富导师"，路人骂"骗子/大忽悠/庄家/割韭菜"。
- 新京报概括其两次"变脸"：新东方名师/畅销书作家（人生导师）→ 比特币首富/天使投资人（韭菜之王）。
- 核心张力（多家媒体共识）：**公开教导**（耐心、长期、"价值投资"、《把时间当作朋友》《通往财富自由之路》）vs **私下言论**（"赚钱才是成功""忽悠不是贬义""傻X共识也是共识"）之间的巨大反差。这正是录音门杀伤力最大的地方——它让"导师人设"与"操盘手实质"短兵相接。
  - 来源：新京报、小宇宙播客《李笑来：比特币首富与争议漩涡》归纳 https://www.xiaoyuzhoufm.com/episode/684248216dbe9284e70399e1

### 4.2 知识付费"贩卖焦虑"批评【二手-权威】

- 2016年7月李笑来在**得到App**开《通往财富自由之路》专栏，售价199元/年，订阅超20万，营收数千万，是现象级知识付费产品。
- 与罗振宇/得到的关系：录音里李笑来自述"罗振宇帮我卖书……跟他合作开专栏"是其影响力放大的关键一环。媒体把李笑来与罗振宇、吴晓波并列为"贩卖焦虑/知识付费"的代表人物。
- 对该模式的批评（针对罗振宇/得到，外溢到李笑来）：
  - "贩卖知识焦虑和成功焦虑来敛财""碎片化伪知识""毒鸡汤""智商税""速效救心丸"（许知远语："把知识放在胶囊里"）。
