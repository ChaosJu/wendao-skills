# Zone 调研目录结构说明

Zone 是**主题 skill**（交易心理学综合体系），不是单一人物 skill。

按女娲「主题 Skill vs 人物 Skill」规范，主题 skill 的调研按**核心人物/流派**分文件组织，而非按人物 skill 的 6 维度（01-writings ~ 06-timeline）组织。因此本目录采用人物分卷命名：

| 文件 | 内容 |
|------|------|
| `00-quality-check.md` | Phase 4 质量验证报告 |
| `01-mark-douglas.md` | Mark Douglas：交易心理学奠基（接受风险、概率思维、五大真理） |
| `02-brett-steenbarger.md` | Brett Steenbarger：交易绩效心理学（刻意练习、情绪管理） |
| `03-elder-dayton.md` | Alexander Elder + Mark Dayton：纪律与三 M 框架 |
| `04-kahneman-cognitive.md` | Kahneman：行为经济学与认知偏差（损失厌恶、前景理论） |
| `05-thaler-housel.md` | Thaler + Housel：心理账户与金钱心理学 |
| `06-timeline.md` | 交易心理学发展时间线与各家关系 |

> 一手来源以各人物原著/演讲为主；评分与媒体转述仅作交叉参考。
